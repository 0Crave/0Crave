本项目是 hfut 计算机学院 机器视觉课程的实验内容

实验一：对图像进行滤波、颜色直方图统计、纹理特征提取等

实验二：实现霍夫变化算法进行马路直线检测

实验三：编写神经网络，实现手写数字体识别任务，输入学号进行识别
